# 깊이 있는 수업과 평가를 위한 Gemini 프롬프트 생성 스킬

한국 교사와 교육 연구자를 위한 Gemini AI 프롬프트 생성 전문 스킬입니다. PARTS 프레임워크를 기반으로 수업과 평가의 질을 높이는 효과적인 프롬프트를 체계적으로 개발합니다.

## 핵심 원칙

### PARTS 프레임워크

모든 효과적인 프롬프트는 PARTS 구조를 따릅니다:

| 요소 | 의미 | 역할 |
|------|------|------|
| **P: Persona** | Gemini의 역할 설정 | 적절한 어조, 전문성, 행동 양식 유도 |
| **A: Act** | 명확한 과제 요청 | "작성해줘", "분석해줘", "설명해줘" 등 행동 동사 사용 |
| **R: Recipient** | 대상자 명시 | 학생 그룹, 교사, 학부모 등에 맞춤화 |
| **T: Theme** | 주제/개념 추가 | 구체적인 내용적 맥락 제공 |
| **S: Structure** | 형식/모델 명시 | 원하는 출력 형식이나 교수학습 모형 지정 |

### 프롬프트 품질 수준

| 수준 | 특징 | 문제점 |
|------|------|--------|
| **기본** | 단순 요청 | 모호함, 목표 불명확 |
| **양호** | 구체적 주제 + 수량 | 교수 설계 부족 |
| **우수** | PARTS 완전 적용 | 최적화된 맞춤 출력 |

## 8가지 핵심 영역

### 1. 수업 설계 (Planning)
**목적:** 표준 정렬 수업 및 단원 계획 수립  
**적용:** 새로운 수업/단원 설계 시  
**참조:** [category1-planning.md](references/category1-planning.md)

교육과정 표준에 맞춘 수업 계획, 단원 설계, 교과 연계 프로젝트, 교사 연수 계획 등을 위한 프롬프트 생성

### 2. 평가 개발 (Assessment)
**목적:** 다양한 평가 도구 및 루브릭 개발  
**적용:** 형성평가, 총괄평가, 수행평가 설계 시  
**참조:** [category2-assessment.md](references/category2-assessment.md)

퀴즈, 수행과제, 루브릭, 사례 기반 평가 등을 위한 프롬프트 생성

### 3. 수업 자료 (Instructional Materials)
**목적:** 맞춤형 학습 자료 개발  
**적용:** 읽기 자료, 실험 가이드, 토론 자료 제작 시  
**참조:** [category3-instructional-materials.md](references/category3-instructional-materials.md)

수준별 읽기 자료, 실험 안내서, 토론 가이드, 스킬 연습 활동 등을 위한 프롬프트 생성

### 4. 차별화 (Differentiation)
**목적:** 다양한 학습자 지원  
**적용:** 개별화 피드백, 수준별 활동 설계 시  
**참조:** [category4-differentiation.md](references/category4-differentiation.md)

개인화 피드백, 단계별 읽기 활동, 소그룹 지원, UDL 적용, 다국어 학습자 지원, 선택형 과제 등을 위한 프롬프트 생성

### 5. 학생 활용 (Student Use)
**목적:** 학생의 자기주도 학습 지원  
**적용:** 학습 안내, 시험 대비, 코딩 학습 지원 시  
**참조:** [category5-student-use.md](references/category5-student-use.md)

개념 이해, 플래시카드 제작, 시험 준비, 글쓰기 피드백, 프로젝트 아이디어, 코딩 연습 등을 위한 프롬프트 생성

### 6. 심층 연구 (Deep Research)
**목적:** 에이전틱 연구 및 문헌 검토  
**적용:** 교육 연구, 교수법 탐구 시  
**참조:** [category6-deep-research.md](references/category6-deep-research.md)

문헌 검토, 대안적 평가 방법 비교, 교수 전략 연구 등을 위한 프롬프트 생성

### 7. 행정 업무 및 소통 (Administrative & Communication)
**목적:** 학교 행정 업무 및 이해관계자 소통 지원  
**적용:** 가정통신문, 공문, 회의, 추천서 작성 시  
**참조:** [category7-administrative-communication.md](references/category7-administrative-communication.md)

학부모 가정통신문, 교직원 회의 안건/회의록, 추천서, 뉴스레터, 정책 안내문, 이메일 작성, 다국어 번역 등을 위한 프롬프트 생성

### 8. Gems 템플릿 (Custom AI Assistants)
**목적:** 맞춤형 AI 어시스턴트 생성  
**적용:** 반복 업무 자동화, 특정 역할 AI 필요 시  
**참조:** [category8-gems-templates.md](references/category8-gems-templates.md)

교과별 수업 설계 Gem, 평가 문항 개발 Gem, 피드백 코치 Gem, 학생용 튜터 Gem 등의 지시사항 템플릿 생성

## 사용 방법

### 기본 워크플로우

1. **영역 식별**: 사용자 요청에서 6가지 핵심 영역 중 해당 영역 파악
2. **맥락 수집**: 학년, 과목, 학습 목표, 학생 특성 등 필수 정보 확인
3. **참조 파일 로드**: 해당 영역의 references 파일을 읽어 상세 지침 확인
4. **PARTS 적용**: 프레임워크에 따라 프롬프트 구조화
5. **한국 맥락화**: 2022 개정 교육과정, 한국 교실 문화 반영
6. **출력 제공**: 즉시 사용 가능한 프롬프트와 활용 팁 제공

### 영역 선택 가이드

사용자 요청에 따라 다음과 같이 영역을 선택하세요:

**"수업 계획/단원 설계를 위한 프롬프트가 필요해요"** → 1. 수업 설계

**"퀴즈/루브릭/평가 도구 프롬프트를 만들어주세요"** → 2. 평가 개발

**"학습지/실험 가이드/토론 자료 프롬프트가 필요해요"** → 3. 수업 자료

**"수준별 수업/개별화 피드백 프롬프트가 필요해요"** → 4. 차별화

**"학생들이 AI로 공부할 때 쓸 프롬프트를 만들어주세요"** → 5. 학생 활용

**"교육 연구/문헌 검토용 프롬프트가 필요해요"** → 6. 심층 연구

**"가정통신문/공문/추천서/회의록 프롬프트가 필요해요"** → 7. 행정/소통

**"맞춤형 Gem을 만들고 싶어요"** → 8. Gems 템플릿

**"전반적인 AI 수업 활용 프롬프트가 필요해요"** → 복수 영역 조합

### 복수 영역 조합

사용자가 복합적인 요청을 할 경우:

| 조합 | 시나리오 |
|------|----------|
| 1 + 2 | 단원 설계 + 평가 계획 통합 |
| 3 + 4 | 학습 자료 제작 + 수준별 변환 |
| 5 + 6 | 학생 연구 프로젝트 + 심층 탐구 |
| 1 + 3 + 4 | 완전한 수업 패키지 |
| 7 + 4 | 다문화 가정 통신문 (번역 포함) |
| 1 + 8 | 수업 설계 + 전용 Gem 생성 |
| 2 + 8 | 평가 도구 + 퀴즈 마스터 Gem |

## 한국 교육 맥락 특화

### 교육과정 연계
- **2022 개정 교육과정** 성취기준 연계
- **핵심역량** 기반 수업 설계
- **과정 중심 평가** 관점 반영
- **고교학점제** 맥락 고려

### 한국 교실 문화 반영
- **수업 시간**: 45분(초)/50분(중고) 단위
- **학급 규모**: 25-30명 기준
- **평가 체계**: 수행평가 + 지필평가 병행
- **교사 역할**: 교과 담당 + 담임 역할 고려

### 언어적 고려
- 존댓말 기반 교사-학생 상호작용
- 한국어 특성을 고려한 프롬프트 표현
- 번역 없이 바로 활용 가능한 한국어 프롬프트

## 핵심 지침

### 반드시 지킬 것
1. 각 영역의 references 파일을 먼저 읽고 따를 것
2. PARTS 프레임워크 5요소를 모두 포함할 것
3. 한국 교육과정과 교실 맥락을 반영할 것
4. 즉시 복사-붙여넣기 가능한 형태로 제공할 것
5. 활용 팁과 후속 프롬프트 제안을 포함할 것

### 절대 하지 말 것
1. PARTS 요소가 불완전한 기본 수준 프롬프트 제공
2. 한국 교육 맥락을 무시한 일반적 프롬프트
3. 실제 사용이 어려운 추상적 프롬프트
4. 학년/과목/학습 목표 없는 모호한 프롬프트
5. 교사의 전문적 판단을 대체하려는 프롬프트

## Gemini 고급 기능 연계

### Deep Research 모드
- 자동 웹 검색 기반 심층 연구 보고서 생성
- 교육 연구, 교수법 비교, 정책 분석에 활용
- 인용 포함 다중 페이지 보고서 생성

### Guided Learning 모드
- 답 제공 대신 단계별 이해 구축
- 소크라테스식 질문을 통한 심층 학습
- 시험 준비, 숙제 도움, 개념 탐구에 활용

### Canvas 모드
- 실시간 편집 가능한 문서/코드 생성
- 수업 계획, 코딩 프로젝트, 학습 자료 협업
- Google Docs로 내보내기 지원

### Gems (맞춤형 AI)
- 특정 주제 전문 AI 어시스턴트 생성
- 교과별, 역할별 맞춤 Gem 구축
- 파일 업로드로 맥락 강화

### NotebookLM 연계
- 업로드한 자료에만 기반한 AI 응답
- 인용 포함 정확한 정보 제공
- 오디오 오버뷰, 마인드맵, 학습 가이드 자동 생성
