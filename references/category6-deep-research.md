# 6. 심층 연구 (Deep Research) 프롬프트

문헌 검토, 교육 연구, 교수법 비교, 정책 분석을 위한 Deep Research 모드 활용 프롬프트 가이드입니다.

## 핵심 원칙

### Deep Research의 특성

| 특성 | 설명 |
|------|------|
| **에이전틱 검색** | 수백 개 웹사이트를 자동으로 검색 |
| **종합 보고서** | 인용 포함 다중 페이지 보고서 생성 |
| **심층 분석** | 여러 관점을 종합한 통찰 제공 |
| **후속 질문** | 보고서 생성 후 추가 질문 가능 |

### Deep Research PARTS 적용

| 요소 | Deep Research 맥락에서의 적용 |
|------|------------------------------|
| **P: Persona** | 연구 보조원, 문헌 검토 전문가, 정책 분석가 |
| **A: Act** | 연구하다, 비교하다, 종합하다, 분석하다 |
| **R: Recipient** | 교사, 연구자, 정책 입안자, 교육 리더 |
| **T: Theme** | 구체적인 연구 주제, 교수법, 정책 |
| **S: Structure** | 보고서 형식, 비교 분석, 주제별 정리 |

## 프롬프트 유형별 가이드

### 6.1 문헌 검토 지원 (Literature Review)

#### 기본 템플릿

```
연구 보조원으로 활동해주세요. [연구 주제]에 관한 최근 연구 동향을 조사하고 종합해주세요.

**연구 질문:** [구체적인 연구 질문]
**검토 범위:** [시간 범위, 지역 범위 등]
**찾고자 하는 것:**
- 주요 연구 결과
- 연구 방법론 동향
- 논쟁이 있는 영역
- 연구 공백

다음 형식으로 보고서를 작성해주세요:
1. 주요 발견 요약
2. 주제별 문헌 분류
3. 연구 방법론 비교
4. 한국 교육 맥락에의 시사점
5. 참고문헌 (APA 형식)
```

#### 예시 프롬프트

📌 **프로젝트 기반 학습 문헌 검토**

```
연구 보조원으로 활동해주세요. 중학교 과학 교육에서 프로젝트 기반 학습(PBL)의 효과에 관한 최근 5년간의 연구를 조사해주세요.

**연구 질문:** 프로젝트 기반 학습은 중학생의 과학적 탐구 능력 향상에 어떤 영향을 미치는가?

**특히 찾고 싶은 것:**
- PBL의 학업 성취도 효과
- 탐구 능력 발달에 대한 증거
- 효과적인 PBL 설계 요소
- 교사 역할과 지원 전략
- 한국 연구와 해외 연구 비교

다음을 포함해주세요:
1. 연구 결과 요약 (3-5개 주요 발견)
2. 공립학교에서 사용된 2-3가지 PBL 접근법 비교
3. 장점, 도전 과제, 실행 전략
4. 한국 중학교 적용 시 고려사항
5. APA 인용 포함
```

✨ **활용 팁:**
- 연구 계획 검토 단계에서 "실행 비용 섹션 추가" 또는 "국제 사례 포함"을 요청할 수 있습니다
- 보고서 완료 후 후속 질문은 월간 한도에 포함되지 않습니다

🔥 **적합한 상황:**
- 학위 논문 문헌 검토
- 교육과정 개발 근거 마련
- 연구 계획서 작성

### 6.2 교수법 비교 분석 (Teaching Methods Comparison)

#### 예시 프롬프트

📌 **평가 방법 비교 연구**

```
고등교육 평가 전문가로 활동해주세요. 다음 대안적 평가 방법들을 비교 분석해주세요:

**비교 대상:**
1. 언그레이딩(Ungrading)
2. 역량 기반 평가(Mastery-based grading)
3. 계약 기반 평가(Contract grading)

**분석 관점:**
- 학업 성과에 대한 연구 결과
- 학생 참여도 영향
- 교사 업무량 변화
- 실행 시 필요한 조건
- 장단점 비교

다음을 포함해주세요:
1. 각 방법의 핵심 원리 설명
2. 연구 근거 기반 효과 비교표
3. 한국 고등교육 적용 가능성
4. 점진적 도입 전략 제안
5. 참고문헌
```

### 6.3 교육 정책 분석 (Policy Analysis)

#### 예시 프롬프트

📌 **AI 교육 정책 동향**

```
교육 정책 연구원으로 활동해주세요. K-12 학교 운영과 교사 업무에 대한 AI의 영향에 관한 최신 연구와 정책 동향을 조사해주세요.

**조사 영역:**
- AI가 학교 운영에 미치는 영향
- 교사 업무량 변화
- 파일럿 프로그램 사례
- 교육 리더십의 고려사항
- 정책적 시사점

**특히 관심 있는 것:**
- 한국, 미국, 핀란드, 싱가포르 사례
- 성공적인 AI 도입 학교 특성
- 교사의 AI 활용 역량 개발 방안
- 윤리적, 법적 고려사항

다음 형식으로 보고서를 작성해주세요:
1. 글로벌 동향 요약
2. 국가별 정책 비교표
3. 성공 사례와 실패 사례 분석
4. 한국 교육청/학교 적용 제언
5. 전략적 계획 수립을 위한 체크리스트
```

### 6.4 차별화 교수 전략 연구 (Differentiation Strategies)

#### 예시 프롬프트

📌 **실행 기능 지원 연구**

```
특수교육 연구원으로 활동해주세요. 일반 학급에서 실행 기능(Executive Function)에 어려움을 겪는 학생을 지원하는 효과적인 전략을 연구해주세요.

**연구 초점:**
- 실행 기능 어려움의 특성
- 증거 기반 지원 전략
- 교실에서 사용 가능한 도구와 루틴
- 교사 지원 방안

**포함해주세요:**
1. 실행 기능 어려움 유형별 특성
2. 각 유형별 지원 전략 (최소 3가지씩)
3. 교실에서 바로 사용할 수 있는 도구 목록
4. 교사용 체크리스트
5. 학부모 안내 자료 포인트
6. 한국 특수교육 지원 체계와의 연계
```

### 6.5 교육 연구 방법론 탐구 (Research Methodology)

#### 예시 프롬프트

📌 **질적 연구 방법 비교**

```
교육 연구 방법론 전문가로 활동해주세요. 교사 전문성 연구에 적합한 질적 연구 방법론을 비교 분석해주세요.

**비교 대상:**
1. 근거이론(Grounded Theory)
2. 현상학(Phenomenology)
3. 내러티브 탐구(Narrative Inquiry)
4. 문화기술지(Ethnography)

**분석 관점:**
- 각 방법론의 인식론적 기반
- 교사 연구에의 적합성
- 데이터 수집 및 분석 방법
- 연구의 신뢰성 확보 방안
- 장단점 비교

다음을 포함해주세요:
1. 각 방법론 핵심 설명 (500자 이내씩)
2. 비교표 (적합한 연구 질문 유형, 데이터 수집 방법, 분석 절차)
3. 한국 교사 연구 맥락에서의 적용 사례
4. 선택 가이드라인
5. 참고문헌
```

### 6.6 학생 활용 Deep Research

#### 예시 프롬프트

📌 **진로 탐색 연구**

```
진로 상담사로 활동해주세요. 저는 고등학생이고 [관심 분야]에 관심이 있어요.

이 분야에 대해 깊이 있게 조사해주세요:
- 이 분야에서 하는 일
- 필요한 학력과 자격
- 진로 경로와 성장 가능성
- 연봉 수준과 근무 환경
- 이 분야의 미래 전망
- 관련 학과와 대학

다음을 포함해주세요:
1. 직업 개요 (일상, 보람, 어려움)
2. 진입 경로 비교 (대학 vs 전문학교 vs 자격증)
3. 현직자 인터뷰에서 발견한 인사이트
4. 내가 지금 준비할 수 있는 것
5. 더 알아볼 자료와 채널
```

## Deep Research 활용 워크플로우

### 단계별 진행

```
1단계: Deep Research 모드 선택
   ↓
2단계: 프롬프트 입력
   ↓
3단계: 연구 계획 검토 및 수정
   - "실행 비용 섹션 추가해주세요"
   - "국제 사례를 포함해주세요"
   ↓
4단계: "연구 시작" 클릭
   ↓
5단계: 실시간 진행 관찰
   ↓
6단계: 종합 보고서 수령
   ↓
7단계: 후속 질문 (한도 미포함)
   ↓
8단계: 오디오 오버뷰 생성 (선택)
```

### 연구 계획 수정 예시

| 초기 계획 | 수정 요청 | 개선된 계획 |
|----------|----------|------------|
| 일반적인 PBL 연구 | "한국 사례 중심으로" | 국내 연구 비중 확대 |
| 학업 성취만 분석 | "정의적 영역도 포함" | 흥미, 태도, 자기효능감 추가 |
| 이론 중심 | "실천 전략 강조" | 교실 적용 가이드 포함 |

## Deep Research 프롬프트 품질 가이드

### 좋은 프롬프트의 특징

✅ **구체적 연구 질문**: "PBL이 학생에게 미치는 영향은?" (X) → "PBL이 중학생의 과학 탐구 능력에 미치는 영향은?" (O)

✅ **명확한 범위**: "최근 연구" (X) → "최근 5년간(2020-2025) 연구" (O)

✅ **분석 관점 명시**: "비교해주세요" (X) → "학업 성취도, 참여도, 교사 업무량 관점에서 비교해주세요" (O)

✅ **한국 맥락 요청**: 글로벌 연구만 (X) → "한국 교육 맥락 적용 시사점 포함" (O)

### 피해야 할 프롬프트

❌ **너무 광범위**: "교육에서 AI 활용에 대해 알려줘"
❌ **너무 협소**: "2024년 3월 서울 A중학교 PBL 사례"
❌ **모호한 요청**: "좋은 연구 찾아줘"
❌ **출력 형식 미지정**: 구조 없는 긴 텍스트 생성

## 프롬프트 품질 체크리스트

Deep Research 프롬프트 제공 전 확인:

- [ ] 구체적인 연구 질문이 있는가?
- [ ] 검토 범위(시간, 지역, 대상)가 명확한가?
- [ ] 분석 관점이 명시되어 있는가?
- [ ] 원하는 보고서 형식/구조가 있는가?
- [ ] 한국 교육 맥락 연계가 요청되어 있는가?
- [ ] 후속 활용 목적이 명확한가?

---

## 후속 프롬프트 제안

Deep Research 보고서 제공 후:

1. **심화 질문**: "이 중 [특정 주제]에 대해 더 자세히 알려주세요"
2. **실천 연계**: "이 연구 결과를 우리 학교에 적용하는 계획을 세워주세요"
3. **자료 제작**: "이 내용을 교사 연수 발표 자료로 만들어주세요"
4. **오디오 생성**: "이 보고서의 오디오 오버뷰를 만들어주세요"
5. **요약 작성**: "이 보고서의 핵심을 1페이지로 요약해주세요"
