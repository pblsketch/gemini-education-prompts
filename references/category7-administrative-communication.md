# 7. 행정 업무 및 소통 (Administrative Tasks & Communication) 프롬프트

학부모 소통, 교직원 회의, 학교 운영, 공문 작성, 추천서 등 행정 업무를 위한 프롬프트 가이드입니다.

## 핵심 원칙

### 행정/소통 프롬프트의 PARTS 적용

| 요소 | 행정/소통 맥락에서의 적용 |
|------|--------------------------|
| **P: Persona** | 학교 커뮤니케이션 담당자, 교무 기획자, 행정 보조 |
| **A: Act** | 작성하다, 요약하다, 기획하다, 번역하다 |
| **R: Recipient** | 학부모, 교직원, 학생, 지역사회, 동문 |
| **T: Theme** | 행사, 정책, 공지, 평가 결과, 추천 내용 |
| **S: Structure** | 가정통신문, 회의록, 뉴스레터, 추천서 형식 |

## 프롬프트 유형별 가이드

### 7.1 학부모 가정통신문 (Parent Communication)

#### 기본 템플릿

```
당신은 학교 커뮤니케이션 전문가입니다. [목적]에 관한 학부모 가정통신문을 작성해주세요.

**발송 대상:** [학년/학급]
**주제:** [구체적인 내용]
**포함 사항:** [필수 전달 내용]
**어조:** [공식적/친근한/긴급한]

다음을 포함해주세요:
- 명확한 제목
- 핵심 내용 요약
- 필요한 조치 사항 (회신, 준비물 등)
- 문의처 안내
```

#### 예시 프롬프트

📌 **현장학습 안내 가정통신문**

```
당신은 중학교 학부모 소통 담당 교사입니다. 2학년 현장학습 안내 가정통신문을 작성해주세요.

**발송 대상:** 중학교 2학년 학부모
**행사 정보:**
- 일시: 2025년 5월 15일(목) 08:30-17:00
- 장소: 국립중앙박물관 + 용산가족공원
- 비용: 15,000원 (교통비, 중식 포함)
- 준비물: 편한 복장, 필기구, 물

**포함 사항:**
- 교육적 목적 설명
- 상세 일정
- 안전 수칙
- 참가 동의서 회신 안내
- 비용 납부 방법
- 우천 시 대체 계획

**어조:** 친근하지만 공식적

※ 한글 파일에 바로 붙여넣을 수 있는 형식으로, 학교 서식에 맞게 작성해주세요.
```

✨ **활용 팁:**
- "학부모 관점에서 궁금할 사항을 먼저 파악하고 답해주세요"를 추가하면 더 완성도 높은 통신문이 됩니다
- 다국어 번역이 필요하면 후속 프롬프트로 요청하세요

🔥 **적합한 상황:**
- 현장학습, 체험활동, 학교 행사 안내
- 새 학기 안내, 방학 안내
- 정책/제도 변경 안내

### 7.2 긴급 공지 및 안전 안내

#### 예시 프롬프트

📌 **비상 상황 안내문**

```
당신은 학교 안전 담당자입니다. 다음 상황에 대한 학부모 긴급 안내문을 작성해주세요.

**상황:** 학교 인근 공사로 인한 임시 등하교 경로 변경
**기간:** 2025년 6월 1일 ~ 6월 30일 (약 1개월)
**변경 내용:**
- 정문 사용 불가, 후문으로 등하교
- 차량 진입로 변경
- 스쿨버스 승하차 위치 변경

**어조:** 차분하지만 명확하게

다음을 포함해주세요:
- 변경 이유 간단히
- 구체적인 변경 내용 (지도 또는 약도 삽입 위치 표시)
- 안전 주의사항
- 학생 지도 협조 요청
- 문의처
```

### 7.3 학업 성취 및 평가 결과 안내

#### 예시 프롬프트

📌 **학기말 성적표 안내문**

```
당신은 학교 교무 담당 교사입니다. 학기말 성적표 배부에 관한 학부모 안내문을 작성해주세요.

**시기:** 1학기 말
**배부 방식:** 학생 편 가정 전달
**포함 내용:**
- 성적표 구성 설명 (교과 성적, 세부능력특기사항, 출결 등)
- 성적표 해석 방법 안내
- 성적 관련 상담 신청 안내
- 2학기 학습 지원 안내

**어조:** 따뜻하고 격려하는 톤

다음을 강조해주세요:
- 성적은 성장의 과정이라는 메시지
- 가정에서의 격려와 지지 요청
- 학교-가정 협력 강조
```

### 7.4 교직원 회의 운영

#### 예시 프롬프트

📌 **교직원 회의 안건 및 진행**

```
당신은 교무 기획 담당 교사입니다. 다음 주제의 교직원 회의 안건을 작성해주세요.

**회의 주제:** AI 활용 수업 도구 도입 논의
**회의 시간:** 45분
**참석 대상:** 전체 교사 (32명)
**회의 목적:** AI 수업 도구 소개 및 도입 의견 수렴

다음을 포함해주세요:
1. 회의 안건표 (시간 배분 포함)
2. 각 안건별 논의 포인트
3. 토론 촉진 질문 3개
4. 의견 수렴 방법 제안
5. 후속 조치 사항 정리 양식
```

📌 **회의록 작성 지원**

```
당신은 회의 기록 담당자입니다. 다음 회의 내용을 바탕으로 공식 회의록을 작성해주세요.

**회의 정보:**
- 회의명: 2025학년도 1학기 교육과정 운영 협의회
- 일시: 2025년 3월 5일(수) 16:00-17:30
- 장소: 대회의실
- 참석자: [참석자 명단]

**논의 내용:** [회의 중 메모 또는 녹음 전사 내용 첨부]

다음 형식으로 정리해주세요:
1. 회의 개요
2. 안건별 논의 내용 요약
3. 결정 사항
4. 후속 조치 및 담당자
5. 차기 회의 안내
```

### 7.5 추천서 작성 (Recommendation Letter)

#### 예시 프롬프트

📌 **대학 입학 추천서**

```
당신은 고등학교 담임교사입니다. 학생의 대학 입학을 위한 추천서 초안을 작성해주세요.

**학생 정보:**
- 이름: [학생명]
- 학년: 고등학교 3학년
- 지원 분야: [전공/계열]
- 지원 대학: [대학명]

**학생 특성 (담임 관찰 내용):**
- 학업적 강점: [구체적 내용]
- 리더십/협력 경험: [구체적 내용]
- 성장 과정: [변화와 발전]
- 인성적 특징: [구체적 에피소드]

**추천 포인트:**
- [강조하고 싶은 점 1]
- [강조하고 싶은 점 2]
- [강조하고 싶은 점 3]

다음을 포함해주세요:
- 추천인과 학생의 관계
- 학업 역량에 대한 평가
- 잠재력과 성장 가능성
- 구체적인 에피소드 기반 서술
- 해당 전공/대학에 적합한 이유

※ 초안이므로 교사가 검토하고 수정할 수 있도록 작성해주세요.
```

📌 **장학금 추천서**

```
당신은 학과 지도교수입니다. 학생의 장학금 신청을 위한 추천서를 작성해주세요.

**장학금 종류:** [장학금명]
**장학금 선발 기준:** [학업 성적, 리더십, 봉사활동, 경제적 필요 등]

**학생 정보:**
- 학업 성취: [GPA, 학업 관련 성과]
- 과외 활동: [동아리, 봉사, 연구 등]
- 경제적 상황: [해당되는 경우]
- 수상/인정: [관련 내용]

다음을 강조해주세요:
- 장학금 선발 기준에 부합하는 구체적 증거
- 학생의 잠재력과 목표
- 장학금이 학생에게 미칠 영향
```

### 7.6 뉴스레터 및 정기 소통

```
당신은 학교 커뮤니케이션 담당자입니다. 월간 학교 뉴스레터 템플릿을 작성해주세요.

**발송 대상:** 학부모, 동문, 지역사회
**포함 섹션:**
1. 교장 인사말
2. 이달의 주요 행사 및 소식
3. 학생 활동 하이라이트
4. 다가오는 일정
5. 학부모 참여 안내

**어조:** 따뜻하고 자랑스러운

다음을 제안해주세요:
- 각 섹션별 샘플 콘텐츠
- 시각적 구성 제안 (사진 배치 등)
- 참여 유도 문구
```

📌 **동문 소식지**

```
당신은 대학 동문회 소통 담당자입니다. 분기별 동문 소식지 템플릿을 작성해주세요.

**포함 섹션:**
1. 학과/학교 소식
2. 동문 성공 사례
3. 기부금 활용 현황
4. 동문회 행사 안내
5. 후배 멘토링 참여 안내

각 섹션에 대해:
- 플레이스홀더 텍스트
- 작성 가이드라인
- 참여/기부 유도 CTA(Call to Action)
```

### 7.7 정책 및 제도 안내

📌 **새로운 정책 안내문**

```
당신은 학교 정책 담당자입니다. 새로운 AI 활용 교육 정책에 관한 교직원 안내문을 작성해주세요.

**정책 내용:**
- 정책명: 교육용 AI 도구 활용 지침
- 시행일: 2025년 9월 1일

다음을 포함해주세요:
- 정책 도입 배경

- 전체 정책 문서 링크

학부모용 버전도 함께 만들어주세요.
```

## 어조 가이드

### 상황별 적절한 어조

| 상황 | 적절한 어조 | 피해야 할 어조 |
|------|------------|---------------|
| **일반 안내** | 친근하고 명확 | 딱딱하고 관료적 |
| **긴급 상황** | 차분하고 명확 | 공포 유발, 모호함 |
| **칭찬/축하** | 따뜻하고 진정성 있게 | 과장되거나 형식적 |
| **문제 상황** | 협력적, 해결 중심 | 비난, 방어적 |
| **공식 문서** | 전문적, 객관적 | 지나치게 캐주얼 |
| **추천서** | 구체적, 진정성 있게 | 일반적, 과장 |

## 다국어 지원

#### 예시 프롬프트

📌 **가정통신문 번역**
