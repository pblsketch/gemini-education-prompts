# PARTS 프레임워크 상세 가이드

효과적인 Gemini 프롬프트 작성을 위한 PARTS 프레임워크 완전 가이드입니다.

## PARTS 프레임워크 개요

PARTS는 Google에서 개발한 교육용 AI 프롬프트 작성 프레임워크로, 5가지 핵심 요소를 통해 일관성 있고 효과적인 프롬프트를 구성합니다.

```
P - Persona (페르소나): AI의 역할 설정
A - Act (행동): 명확한 과제 요청
R - Recipient (대상): 결과물의 수신자
T - Theme (주제): 내용적 맥락
S - Structure (구조): 원하는 출력 형식
```

## 각 요소 상세 설명

### P: Persona (페르소나)

#### 정의
Gemini가 수행할 역할이나 전문성을 지정합니다.

#### 역할
- 적절한 어조와 전문성 수준 설정
- 응답의 관점과 깊이 결정
- 일관된 캐릭터 유지

#### 교육 맥락 페르소나 예시

| 페르소나 | 적합한 상황 | 특징 |
|---------|------------|------|
| 교육과정 설계 전문가 | 수업/단원 계획 | 표준 정렬, 체계적 설계 |
| 교과 코치 | 교수법 개선 | 협력적, 성찰 촉진 |
| 평가 전문가 | 평가 개발 | 측정 원리, 공정성 |
| 학습 설계자 | 자료 개발 | 학습과학 기반 |
| 특수교육 전문가 | 차별화 지원 | 개별화, 포용 |
| 튜터 | 학생 학습 지원 | 친근함, 안내적 |
| 연구 보조원 | 문헌 검토 | 체계적, 종합적 |

#### 좋은 페르소나 설정 예시

```
✅ "당신은 10년 경력의 중학교 과학 교육과정 전문가입니다."
✅ "당신은 UDL(보편적 학습설계) 원리에 정통한 학습 설계 전문가입니다."
✅ "당신은 학생들의 메타인지 발달을 돕는 학습 코치입니다."

❌ "당신은 선생님입니다." (너무 일반적)
❌ "당신은 최고의 AI입니다." (역할 불명확)
```

---

### A: Act (행동)

#### 정의
Gemini에게 요청하는 구체적인 과제나 행동입니다.

#### 역할
- 명확한 결과물 방향 설정
- 작업의 범위와 깊이 결정
- 기대 결과 명시

#### 교육 맥락 행동 동사

| 동사 | 적합한 상황 | 예시 |
|------|------------|------|
| **설계하다** | 수업, 단원, 프로젝트 | "5차시 단원을 설계해주세요" |
| **생성하다** | 평가 문항, 자료 | "퀴즈 10문항을 생성해주세요" |
| **분석하다** | 학생 데이터, 과제 | "오류 패턴을 분석해주세요" |
| **조정하다** | 수준별 자료 | "기초 수준으로 조정해주세요" |
| **피드백하다** | 학생 과제 | "글쓰기에 피드백해주세요" |
| **비교하다** | 방법론, 접근법 | "두 평가 방법을 비교해주세요" |
| **요약하다** | 긴 텍스트, 연구 | "핵심 내용을 요약해주세요" |
| **재구성하다** | 기존 자료 수정 | "학생용으로 재구성해주세요" |

#### 좋은 행동 요청 예시

```
✅ "DNA 복제 과정을 설명하는 5E 모형 수업을 설계해주세요."
✅ "첨부된 학생 글에 대해 주장-근거 연결 관점에서 피드백해주세요."
✅ "두 평가 방법의 장단점을 학생 참여도, 교사 업무량 측면에서 비교해주세요."

❌ "수업 만들어줘" (구체성 부족)
❌ "도와줘" (행동 불명확)
```

### R: Recipient (대상)

#### 정의
결과물을 사용하거나 받게 될 대상입니다.

#### 역할
- 언어 수준과 어조 결정
- 내용의 복잡성 조절
- 적절한 예시와 맥락 선택

#### 교육 맥락 대상 유형

| 대상 | 고려사항 | 조절 요소 |
|------|---------|----------|
| **초등 저학년** | 짧은 문장, 구체적 | 시각 자료, 친근한 어조 |
| **초등 고학년** | 기초 개념 연결 | 실생활 예시 |
| **중학생** | 추상적 사고 시작 | 논리적 구조 |
| **고등학생** | 비판적 사고 | 심화 내용, 연계 |
| **교사** | 전문 용어 가능 | 실용적 적용 |
| **학부모** | 전문 용어 최소화 | 이해 중심 |
| **교육 리더** | 정책적 시각 | 데이터 기반 |

#### 좋은 대상 명시 예시

```
✅ "사전 지식이 다양한 중학교 2학년 학생들을 위해"
✅ "과학 실험에 관심이 높지만 글쓰기에 어려움을 겪는 학생들을 위해"
✅ "AI 활용 경험이 없는 교사들을 위해"
✅ "영어 학습자(ELL)가 포함된 다문화 학급을 위해"

❌ "학생들을 위해" (구체성 부족)
❌ "모든 사람을 위해" (대상 불명확)
```

---

### T: Theme (주제)

#### 정의
프롬프트의 내용적 맥락과 핵심 개념입니다.

#### 역할
- 응답의 내용 범위 한정
- 관련 지식과 연결
- 구체적 사례와 예시 유도

#### 교육 맥락 주제 설정

| 수준 | 예시 | 구체성 |
|------|------|--------|
| **너무 넓음** | "과학" | ❌ |
| **적절함** | "DNA 구조와 기능" | ✅ |
| **더 구체적** | "DNA 이중 나선 구조의 상보적 염기쌍" | ✅✅ |

#### 주제와 성취기준 연결

```
✅ 좋은 예시:
"주제: DNA 복제 과정
성취기준: [9과08-01] 유전 물질이 DNA임을 알고, DNA의 구조적 특성을 이해한다.
핵심 개념: 헬리케이스, DNA 중합효소, 반보존적 복제"

❌ 피해야 할 예시:
"주제: 유전학" (너무 넓음)
"주제: DNA" (구체성 부족)
```

---

### S: Structure (구조)

#### 정의
원하는 출력의 형식, 모델, 프레임워크입니다.

#### 역할
- 결과물의 형태 결정
- 일관된 포맷 유지
- 활용 용이성 확보

#### 교육 맥락 구조 유형

| 구조 유형 | 적합한 상황 | 특징 |
|----------|------------|------|
| **5E 모형** | 탐구 수업 | 참여-탐색-설명-정교화-평가 |
| **백워드 설계** | 단원 계획 | 목표-평가-활동 역순 |
| **UDL 프레임** | 포용적 수업 | 참여-표상-표현 |
| **점진적 해제** | 직접 교수 | I do-We do-You do |
| **분석적 루브릭** | 평가 기준 | 영역별 수준 기술 |
| **Bloom 분류** | 평가 문항 | 인지 수준별 구분 |
| **SWOT 분석** | 정책/프로그램 평가 | 강점-약점-기회-위협 |

#### 좋은 구조 명시 예시

```
✅ "5E 탐구학습 모형으로 구성하고, 각 단계별로 소요 시간과 핵심 활동을 명시해주세요."
✅ "4수준 분석적 루브릭으로, 각 수준에 구체적인 행동 지표와 예시를 포함해주세요."
✅ "표 형식으로 정리하고, 한글 파일에 바로 붙여넣을 수 있게 해주세요."

❌ "좋은 형식으로" (구체성 부족)
❌ "알아서" (구조 불명확)
```

## PARTS 완성 예시

### 예시 1: 수업 설계

```
[P] 당신은 중학교 과학 교육과정 설계 전문가입니다.

[A] 5차시 단원을 설계해주세요.

[R] 사전 지식이 다양한 중학교 2학년 학생들을 위해

[T] DNA 구조와 기능에 관한 단원으로, 
    성취기준 [9과08-01]에 맞추어

[S] 5E 탐구학습 모형을 적용하고, 
    각 차시별 학습 목표, 핵심 활동, 형성평가를 
    표 형식으로 정리해주세요.
```

**완성된 프롬프트:**

```
당신은 중학교 과학 교육과정 설계 전문가입니다. 사전 지식이 다양한 중학교 2학년 학생들을 위해 DNA 구조와 기능에 관한 5차시 단원을 설계해주세요.

성취기준: [9과08-01] 유전 물질이 DNA임을 알고, DNA의 구조적 특성을 이해한다.

5E 탐구학습 모형을 적용하고, 각 차시별로 다음을 표 형식으로 정리해주세요:
- 학습 목표
- 핵심 활동 (교사 활동 / 학생 활동)
- 형성평가 방법
- 준비물
```

---

### 예시 2: 평가 개발

```
[P] 당신은 국어과 수행평가 설계 전문가입니다.

[A] 논술문 수행평가 과제와 루브릭을 개발해주세요.

[R] 기초 학력이 다양한 고등학교 1학년 학생들을 위해

[T] 비평적 읽기와 글쓰기 단원의 
    논증적 글쓰기 역량 평가를 위해

[S] 학생용 과제 안내문과 
    4수준 분석적 루브릭 형식으로 제공하고,
    자기 평가 체크리스트도 포함해주세요.
```

---

### 예시 3: 차별화 지원

```
[P] 당신은 특수교육 및 차별화 교수 전문가입니다.

[A] 수준별 활동 세트를 설계해주세요.

[R] 기초, 표준, 심화 수준의 학습자가 모두 있는 
    중학교 1학년 수학 학급을 위해

[T] 일차방정식의 실생활 적용 문제 해결 활동으로

[S] 3단계(기초-표준-심화) 활동으로 구분하고,
    각 수준별 지원 요소와 도전 요소,
    공통 학습 목표, 활동 시간을 표로 정리해주세요.
```

## PARTS 품질 진단표

프롬프트 작성 후 각 요소를 점검하세요:

| 요소 | 점검 질문 | 체크 |
|------|----------|------|
| **P** | 역할이 구체적이고 과제에 적합한가? | □ |
| **A** | 행동 동사가 명확하고 구체적인가? | □ |
| **R** | 대상의 특성(학년, 수준, 특성)이 명시되어 있는가? | □ |
| **T** | 주제가 구체적이고 성취기준과 연결되어 있는가? | □ |
| **S** | 원하는 출력 형식이 명확히 요청되어 있는가? | □ |

### 점수별 가이드

- **5개 모두 체크**: 우수한 프롬프트
- **3-4개 체크**: 양호, 부족한 요소 보완 필요
- **1-2개 체크**: 개선 필요, 요소별 보완

```
❌ "DNA에 대해 도와줘"

```
❌ "학생들을 위한 퀴즈"
