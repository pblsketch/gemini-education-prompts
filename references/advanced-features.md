# Gemini 고급 기능 활용 가이드

Gemini의 고급 기능(Deep Research, Guided Learning, Canvas, Gems)을 교육에 활용하는 방법을 안내합니다.

## 1. Deep Research 모드

### 개요

Deep Research는 Gemini의 에이전틱 연구 기능으로, 자동으로 수백 개의 웹사이트를 검색하고 인용이 포함된 종합 보고서를 생성합니다.

### 활성화 방법

1. 채팅창 하단의 "Deep Research" 선택
2. 또는 모델 선택기에서 "Deep Research" 선택

### 워크플로우

```
1. 프롬프트 입력
   ↓
2. 연구 계획 검토
   - 수정 요청: "실행 비용 섹션 추가", "국제 사례 포함"
   ↓
3. "연구 시작" 클릭
   ↓
4. 실시간 진행 관찰
   ↓
5. 종합 보고서 수령
   ↓
6. 후속 질문 (한도 미포함)
   ↓
7. 오디오 오버뷰 생성 (선택)
```

### 교육 활용 사례

| 활용 영역 | 예시 프롬프트 |
|----------|-------------|
| **수업 설계** | "중등 교육에서 효과적인 학제간 교수 모델 연구" |
| **평가 개발** | "고등교육의 대안적 평가 시스템 비교: 언그레이딩, 역량 기반, 계약 기반" |
| **차별화** | "실행 기능 어려움 학생을 위한 효과적인 지원 전략 연구" |
| **연구 지원** | "프로젝트 기반 학습이 과학 탐구 능력에 미치는 영향" |
| **학생 활용** | "데이터 과학 분야 진로 탐색" |

### 오디오 오버뷰

- 보고서 완료 후 팟캐스트 스타일 요약 생성 가능
- 다운로드하여 오프라인 청취
- PLC, 교사 연수 자료로 공유 가능

## 2. Guided Learning 모드

### 개요

Guided Learning은 답을 직접 제공하는 대신 단계별 이해 구축을 돕는 모드입니다. 소크라테스식 질문을 통해 심층 학습을 촉진합니다.

### 활성화 방법

1. 채팅창 하단의 "Guided Learning" 선택 (모바일: "Learning")

### 특징

- 기초 개념부터 시작하여 점진적 설명
- 탐색적 질문으로 비판적 사고 유도
- 학습자의 응답에 맞춰 설명 조정

### 교육 활용 사례

| 활용 상황 | 효과 |
|----------|------|
| **숙제 도움** | 답 대신 문제 해결 과정 안내 |
| **시험 준비** | 개념 이해 확인 및 심화 |
| **새로운 주제 학습** | 단계적 이해 구축 |
| **호기심 탐구** | 깊이 있는 주제 탐색 |

### 예시 프롬프트 (학생용)

```
10.5 kg 로켓이 수직으로 발사됩니다. 연료가 모두 타면 1925 J의 운동에너지를 갖습니다. 로켓은 추가로 얼마나 높이 올라갈까요?

(Guided Learning 모드에서 입력하면 Gemini가 단계별로 안내합니다)
```

### 교사 활용 팁

- 학생들에게 Guided Learning 모드 사용법 안내
- "답을 바로 달라고 하지 말고, 이해를 돕도록 요청하라" 교육
- 자기주도 학습 도구로 활용

## 3. Canvas 모드

### 개요

Canvas는 실시간 편집 가능한 작업 공간으로, 문서와 코드를 협업하여 제작할 수 있습니다.

### 활성화 방법

1. 채팅창 하단의 "Canvas" 선택

### 특징

- 채팅과 별도의 편집 가능한 패널
- 실시간 수정 및 Gemini와 협업
- Google Docs로 내보내기 지원
- 코드 실행 및 테스트 가능

### 교육 활용 사례

| 활용 영역 | 활동 예시 |
|----------|----------|
| **수업 설계** | 수업 계획서 실시간 작성 및 수정 |
| **수업 자료** | 학습지, 토론 가이드 협업 제작 |
| **평가** | 루브릭 실시간 편집 |
| **학생 활용** | 코딩 프로젝트 실행 및 디버깅 |

### 예시 워크플로우

```
1. Canvas 모드 활성화
   ↓
2. 초안 프롬프트 입력
   "중학교 과학 물질의 상태 변화 학습지 초안을 만들어줘"
   ↓
3. Canvas에서 직접 편집
   - 질문 추가/삭제
   - 형식 조정
   ↓
4. Gemini에게 수정 요청
   "3번 문항을 더 쉽게 바꿔줘"
   ↓
5. Google Docs로 내보내기
```

## 4. Gems (맞춤형 AI)

### 개요

Gems는 특정 주제나 역할에 맞춰 설정된 맞춤형 Gemini입니다. 반복되는 프롬프트 설정을 저장하고 재사용할 수 있습니다.

### 생성 방법

1. Gem 관리자 열기
2. "새 Gem 만들기" 선택
3. 이름, 지시사항, 지식 파일 설정
4. 저장

### Gem 설정 요소

| 요소 | 설명 | 예시 |
|------|------|------|
| **이름** | Gem 식별 이름 | "표준 기반 수업 설계사" |
| **지시사항** | 역할, 행동, 규칙 | "당신은 표준 정렬 수업을 설계합니다..." |
| **지식 파일** | 참조할 문서 | 교육과정 문서, 기존 수업 계획 |

### 교육용 Gem 예시

#### 1) 표준 기반 수업 설계사

```
이름: 표준 기반 수업 설계사

지시사항:
당신은 표준 정렬 수업 설계를 지원하는 교육과정 설계 전문가입니다. 
학년 수준 성취기준에 맞춘 수업 템플릿을 작성합니다.
명확한 학습 목표, 정렬된 교수 활동, 형성평가를 포함합니다.
과목 영역이나 학년 수준에 따라 조정합니다.

지식 파일:
- 2022 개정 교육과정 성취기준 문서
- 기존 승인된 수업 계획 예시
- 학교 교육과정 운영 지침
```

#### 2) 교수 자료 번역기

```
이름: 교수 자료 번역기

지시사항:
수업 자료, 과제, 가정통신문을 [언어]로 번역합니다.
가족과 학생에게 적합한 명확하고 전문적인 언어를 사용합니다.
원본의 어조와 형식을 유지합니다.
문화적으로 존중하고 이해하기 쉽게 번역합니다.

지식 파일:
- 채점 및 피드백 용어 샘플
- 학교 통신문 템플릿
```

#### 3) 수업 게임화 도우미

```
이름: 수업 게임화 도우미

지시사항:
수업 내용을 약 10분 분량의 짧고 재미있는 5인 모둠 게임으로 변환합니다.
퀴즈, 팀 도전, 창의적 시나리오 플레이 형식을 우선합니다.
학생 친화적 언어를 사용하고 학년 수준에 맞게 어조를 조정합니다.

지식 파일:
- 현재 학습 주제/단원
- 이전에 사용했거나 좋아했던 게임 형식
```

#### 4) 과제 루브릭 작성기

```
이름: 과제 루브릭 작성기

지시사항:
특정 학습 결과에 맞춘 과제 프롬프트와 루브릭을 작성합니다.
전문적이고 학술적인 어조로 학부/대학원 학생에게 적합하게 작성합니다.

지식 파일:
- 강의계획서 및 학습 결과 문서
- 기존 승인된 루브릭 예시
- 기관 채점 정책 또는 평가 가이드라인
```

### 프리메이드 Gems 활용

Google에서 제공하는 기본 Gems:

| Gem | 용도 |
|-----|------|
| **Learning coach** | 학습 이해 지원 |
| **Career guide** | 진로 탐색 지원 |
| **Brainstormer** | 아이디어 발상 |

## 5. Gemini in Classroom

### 개요

Google Classroom에 통합된 Gemini 도구로, 수업 계획, 차별화, 학생 학습 경험 생성을 지원합니다.

### 주요 기능

#### 스캐폴드 도구 (Scaffolded Tools)

- 컬러 아이콘으로 표시된 가이드형 템플릿
- 수업 계획, 어휘 목록, 활동 등 빠른 생성
- 학년, 주제, 학습 목표 입력으로 맞춤 결과

#### Gemini 스타터 프롬프트

- 미리 작성된 프롬프트로 빠른 시작
- 실제 세계 예시 브레인스토밍, 이메일 초안 등
- Gemini 앱에서 계속 대화 가능

#### NotebookLM/Gems 연결

- 학급 자료 기반 AI 학습 경험 생성
- 학생용 스터디 가이드, 퀴즈 Gem 제공
- 과제에 첨부하여 맞춤형 학습 지원

### 학생 AI 학습 경험

```
교사가 할 수 있는 것:
1. NotebookLM 노트북을 과제에 첨부
2. 학생용 Gem(예: Quiz me, Study partner) 생성 및 첨부
3. 클래스 상단에 학습 도구로 고정
4. 학생 접근 현황 모니터링

학생이 경험하는 것:
- 교사가 제공한 자료에만 기반한 AI 응답
- 맞춤형 학습 지원
- 소크라테스식 학습 안내
```

## 통합 활용 시나리오

### 시나리오 1: 새 단원 준비

```
1. Deep Research로 교수법 동향 조사
2. Canvas에서 단원 계획 작성
3. Gems(수업 설계사)로 차시별 활동 생성
4. NotebookLM에 자료 업로드 및 학생용 스터디 가이드 생성
5. Classroom에서 학생에게 제공
```

### 시나리오 2: 학생 자기주도 학습

```
1. 교사: NotebookLM 노트북에 단원 자료 업로드
2. 교사: Quiz me Gem 생성 및 과제 첨부
3. 학생: NotebookLM에서 개념 질문
4. 학생: Guided Learning에서 어려운 문제 안내받기
5. 학생: Gem으로 맞춤형 퀴즈 연습
```

### 시나리오 3: 교사 연구 및 개발

```
1. Deep Research로 특정 교수 전략 연구
2. 오디오 오버뷰로 PLC 공유 자료 생성
3. Gems로 연구 결과 기반 수업 도구 제작
4. Canvas에서 연수 자료 협업 작성
```
