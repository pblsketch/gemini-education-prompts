# 4. 차별화 (Differentiation) 프롬프트

개인화 피드백, 수준별 활동, 특수 학습자 지원, UDL 적용을 위한 프롬프트 가이드입니다.

## 핵심 원칙

### 차별화 프롬프트의 PARTS 적용

| 요소 | 차별화 맥락에서의 적용 |
|------|----------------------|
| **P: Persona** | 학습 코치, 특수교육 전문가, 개별화 교육 설계자 |
| **A: Act** | 피드백하다, 조정하다, 지원하다, 확장하다 |
| **R: Recipient** | 특정 학습 특성/수준/요구를 가진 학생 |
| **T: Theme** | 학습 내용, 기능, 지원이 필요한 영역 |
| **S: Structure** | 피드백 형식, 지원 단계, UDL 프레임워크 |

## 프롬프트 유형별 가이드

### 4.1 개인화 피드백 (Personalized Feedback)

#### 기본 템플릿

```
당신은 [학년] [교과] 학습 코치입니다. 다음 학생 과제에 대해 개인화된 피드백을 작성해주세요.

**과제 정보:**
- 과제명: [과제명]
- 학습 목표: [평가하는 학습 목표]
- 학생 수준: [상/중/하 또는 구체적 특성]

**학생 과제:** [첨부 또는 입력]

다음 형식으로 피드백해주세요:
1. 긍정적 측면 2가지
2. 개선이 필요한 구체적 영역
3. 구체적인 개선 방법/전략
4. 다음 단계 제안
```

#### 예시 프롬프트

📌 **글쓰기 개인화 피드백**

```
당신은 중학교 국어 학습 코치입니다. 다음 학생의 논술문에 대해 개인화된 피드백을 작성해주세요.

**과제 정보:**
- 과제명: 환경 보호를 위한 제안문 쓰기
- 학습 목표: 
  (1) 주장과 근거의 논리적 연결
  (2) 설득력 있는 표현 사용
- 학생 특성: 아이디어는 풍부하나 문장 구성에 어려움을 겪는 학생

**학생 과제:** [학생 글 첨부]

다음 형식으로 피드백해주세요:
1. 🌟 잘한 점 2가지 (구체적 문장 인용)
2. 📝 개선 포인트 (주장-근거 연결 관련)
3. 💡 구체적 개선 전략 (문장 구조 템플릿 제공)
4. 🚀 다음 글쓰기 도전 과제

※ 학생이 직접 읽을 수 있도록 친근한 어조로 작성해주세요.
```

✨ **활용 팁:**
- 학생 특성을 구체적으로 명시하면 맞춤형 피드백이 생성됩니다
- "수정이 아닌 가이드"를 강조하면 학생 스스로 수정하도록 안내합니다

🔥 **적합한 상황:**
- 개별 학생의 글쓰기나 과제에 맞춤형 피드백이 필요할 때
- 시간 절약하면서도 질 높은 피드백을 제공하고 싶을 때

### 4.2 수준별 활동 설계 (Tiered Activities)

#### 기본 템플릿

```
당신은 수준별 학습 설계 전문가입니다. [주제] 학습을 위한 3단계 수준별 활동을 설계해주세요.

**학습 목표:** [모든 수준에 공통된 학습 목표]
**대상:** [학년]
**활동 유형:** [탐구/연습/프로젝트/토론]

다음 수준별로 설계해주세요:
- 기초: [지원이 필요한 학생]
- 표준: [학년 수준에 도달한 학생]
- 심화: [추가 도전이 필요한 학생]

각 수준에 포함:
- 활동 설명
- 필요한 지원/도전 요소
- 예상 소요 시간
- 성공 기준
```

#### 예시 프롬프트

📌 **수학 수준별 문제 해결 활동**

```
당신은 수준별 수학 학습 설계 전문가입니다. '일차방정식의 활용' 학습을 위한 3단계 수준별 활동을 설계해주세요.

**학습 목표:** 실생활 상황을 일차방정식으로 모델링하고 해결할 수 있다
**대상:** 중학교 1학년
**활동 유형:** 문제 해결 + 모둠 협력

**기초 수준 (지원 필요):**
- 구조화된 문제 (식 세우기 틀 제공)
- 시각적 보조 자료
- 단계별 힌트 카드

**표준 수준 (학년 수준):**
- 다양한 맥락의 실생활 문제
- 자기 점검 체크리스트
- 동료 설명 활동

**심화 수준 (도전 필요):**
- 복잡한 조건의 문제
- 자신만의 문제 만들기
- 여러 가지 해결 전략 비교

각 수준에 포함해주세요:
- 구체적인 활동 설명 및 예시 문제
- 필요한 학습 지원 또는 도전 요소
- 예상 소요 시간
- 자기 평가 기준
```

### 4.3 소그룹 지원 계획 (Small Group Support)

#### 예시 프롬프트

📌 **기초 학력 소그룹 지원**

```
당신은 중학교 국어 학습 지원 전문가입니다. 읽기 이해에 어려움을 겪는 학생들을 위한 3일 소그룹 지원 계획을 수립해주세요.

**대상:** 중학교 2학년 기초 학력 학생 5명
**지원 영역:** 글의 중심 내용 파악
**현재 수준:** 글의 주제를 찾는 데 어려움, 세부 정보와 핵심 정보 구분 어려움
**지원 시간:** 3일 × 30분 (수업 전 또는 점심시간)

다음을 포함해주세요:
1. 진단: 구체적인 어려움 분석
2. 1일차: 시각적 조직자를 활용한 주제 찾기 연습
3. 2일차: 핵심 문장 찾기 전략 학습
4. 3일차: 실제 교과서 글에 전략 적용
5. 각 일차별: 활동 설명, 사용 자료, 성공 기준
6. 일반 수업 복귀를 위한 전략 카드
```

### 4.4 UDL 적용 수업 설계 (Universal Design for Learning)

#### 기본 템플릿

```
당신은 UDL(보편적 학습 설계) 전문가입니다. 다음 수업을 UDL 원리에 따라 재설계해주세요.

**기존 수업:** [수업 내용 설명 또는 첨부]
**대상:** [학년/학급 특성]
**고려할 학습자 다양성:** [인지적/언어적/감각적/신체적]

UDL 3원칙에 따라 설계해주세요:
1. 참여(Engagement): 다양한 동기 유발 방법
2. 표상(Representation): 다양한 정보 제시 방법
3. 행동과 표현(Action & Expression): 다양한 학습 표현 방법
```

#### 예시 프롬프트

📌 **UDL 기반 과학 수업 재설계**

```
당신은 UDL(보편적 학습 설계) 전문가입니다. 다음 과학 수업을 UDL 원리에 따라 재설계해주세요.

**기존 수업:** 
- 주제: 물질의 상태 변화
- 활동: 교사 설명 → 실험 시연 → 학습지 작성
- 평가: 개념 이해 퀴즈

**대상:** 중학교 1학년 (포함 학급)
**학습자 다양성:**
- 주의집중에 어려움을 겪는 학생 2명
- 한국어 학습자(다문화) 학생 1명
- 시각 정보 처리가 강한 학생들
- 손 조작 활동을 선호하는 학생들

**UDL 3원칙에 따라 재설계해주세요:**

1. **참여(Engagement)**
   - 선택권 제공 방안
   - 관련성 높이는 방법
   - 자기조절 지원

2. **표상(Representation)**
   - 다양한 정보 제시 방법 (시각, 청각, 조작)
   - 어휘 지원 방안
   - 핵심 개념 강조 방법

3. **행동과 표현(Action & Expression)**
   - 다양한 반응 방법
   - 표현 도구 선택권
   - 실행 기능 지원

각 영역별로 구체적인 활동과 자료를 제안해주세요.
```

### 4.5 다국어 학습자 지원 (Multilingual Support)

#### 예시 프롬프트

📌 **다문화 학생 학습 지원**

```
당신은 다문화 학생 교육 지원 전문가입니다. 중학교 사회 수업에서 한국어 학습자(KSL) 학생들을 지원하기 위한 전략을 개발해주세요.

**수업 주제:** 민주주의와 시민 참여
**대상 학생:** 
- 중학교 2학년 KSL 학생 3명
- 한국 거주 기간: 1-3년
- 일상 대화 가능, 학습 한국어 어려움

다음을 포함해주세요:
1. **어휘 지원**
   - 핵심 학습 어휘 목록 (10개)
   - 쉬운 정의와 예문
   - 시각적 어휘 카드 형식

2. **내용 이해 지원**
   - 본국의 유사 제도 연결 (베트남, 중국, 필리핀 등)
   - 그림, 도표를 활용한 개념 설명
   - 문장 틀(sentence frame) 제공

3. **수업 참여 지원**
   - 짝 활동 가이드
   - 발표용 문장 패턴
   - 학습 내용 모국어 요약 키워드

4. **평가 조정**
   - 시간 연장 고려
   - 시각 자료 활용 허용
   - 구술 평가 옵션
```

### 4.6 선택형 과제 설계 (Choice Board)

#### 예시 프롬프트

📌 **학습 스타일별 선택판**

```
당신은 차별화 교육 설계 전문가입니다. '기후 변화' 단원의 6옵션 선택판을 만들어주세요.

**학습 목표:** 기후 변화의 원인, 영향, 대응 방안을 설명할 수 있다
**대상:** 중학교 3학년
**과제 형식:** 6가지 옵션 (동일한 학습 목표, 다른 표현 방법)

다음 유형을 포함해주세요:
1. 글쓰기형: 기후 변화 해결 제안서
2. 시각형: 인포그래픽 제작
3. 구술형: 3분 프레젠테이션
4. 창작형: 기후 변화 공익광고 스토리보드
5. 협력형: 모둠 토론 + 합의문 작성
6. 디지털형: 웹사이트 또는 영상 제작

각 옵션에 포함해주세요:
- 과제 설명 (학생용)
- 성공 기준 (3-4개)
- 필요한 도구/자료
- 추천 학생 유형 표시

모든 옵션에 적용되는 공통 루브릭도 만들어주세요.
```

## 차별화 수준 가이드

### 학습 내용별 차별화

| 차별화 요소 | 기초 | 표준 | 심화 |
|------------|------|------|------|
| **복잡성** | 단순, 구조화 | 학년 수준 | 복잡, 다층적 |
| **추상성** | 구체적, 시각적 | 균형 | 추상적, 개념적 |
| **자원** | 지원 자료 다수 | 표준 자원 | 확장 자원 |
| **속도** | 충분한 시간 | 표준 시간 | 압축/가속 |
| **독립성** | 교사 지원 많음 | 적절한 지원 | 자기주도 |

### 학습자 특성별 고려사항

| 학습자 유형 | 지원 전략 |
|------------|----------|
| **느린 학습자** | 시간 연장, 단계 분리, 시각 보조 |
| **빠른 학습자** | 심화 과제, 멘토 역할, 독립 프로젝트 |
| **주의력 어려움** | 짧은 활동, 움직임 포함, 명확한 구조 |
| **언어 장벽** | 어휘 지원, 번역 도구, 시각 자료 |
| **특수교육 대상** | IEP 연계, 보조공학, 환경 조정 |

## 프롬프트 품질 체크리스트

차별화 프롬프트 제공 전 확인:

- [ ] 학습자 특성/수준이 구체적으로 명시되어 있는가?
- [ ] 동일한 학습 목표를 유지하면서 조정하는가?
- [ ] 지원과 도전의 균형이 적절한가?
- [ ] 학생 존엄성을 유지하는 표현을 사용하는가?
- [ ] 실제 적용 가능한 구체적 전략인가?
- [ ] 한국 교실 맥락(학급 규모, 시간 제약)이 반영되어 있는가?

---

## 후속 프롬프트 제안

차별화 프롬프트 제공 후 다음을 제안:

1. **평가 조정**: "이 활동의 수준별 평가 기준을 만들어주세요"
2. **학부모 소통**: "이 차별화 전략을 학부모에게 설명하는 안내문을 작성해주세요"
3. **진도 점검**: "학생 진도를 확인할 체크리스트를 만들어주세요"
4. **일반 수업 연계**: "이 지원 활동을 일반 수업과 연결하는 방법을 제안해주세요"
