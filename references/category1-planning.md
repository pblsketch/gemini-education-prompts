# 1. 수업 설계 (Planning) 프롬프트

수업 및 단원 계획, 교육과정 정렬, 교과 연계 프로젝트, 교사 연수 설계를 위한 프롬프트 가이드입니다.

## 핵심 원칙

### 수업 설계 프롬프트의 PARTS 적용

| 요소 | 수업 설계 맥락에서의 적용 |
|------|--------------------------|
| **P: Persona** | 교육과정 설계자, 교수설계 전문가, 교과 코치 |
| **A: Act** | 설계하다, 계획하다, 개발하다, 정렬하다 |
| **R: Recipient** | 특정 학년/수준의 학생, 학급 특성 |
| **T: Theme** | 구체적인 학습 주제, 성취기준, 핵심 개념 |
| **S: Structure** | 5E 모형, 백워드 설계, 프로젝트 기반, UDL |

## 프롬프트 유형별 가이드

### 1.1 수업 계획 (Lesson Creation)

#### 기본 템플릿

```
당신은 [학교급] [교과] 교육과정 설계 전문가입니다. [학년] 학생들을 위한 [차시] 수업을 설계해주세요.

**학습 주제:** [구체적인 주제/개념]
**성취기준:** [관련 성취기준 코드 및 내용]
**수업 모형:** [적용할 교수학습 모형]

다음을 포함해주세요:
- 학습 목표 (3개 이내)
- 도입-전개-정리 단계별 활동
- 형성평가 방법
- 필요한 학습 자료

수업 시간은 [분]입니다.
```

#### 예시 프롬프트

📌 **5E 모형 기반 과학 수업 설계**

```
당신은 중학교 과학 교육과정 설계 전문가입니다. 중학교 2학년 학생들을 위한 2차시 수업을 5E 모형으로 설계해주세요.

**학습 주제:** DNA 구조와 기능
**성취기준:** [9과08-01] 유전 물질이 DNA임을 알고, DNA의 구조적 특성을 이해한다.
**수업 모형:** 5E 탐구학습 모형

각 단계(참여-탐색-설명-정교화-평가)별로 다음을 포함해주세요:
- 핵심 활동 설명
- 발문 예시
- 학생 참여 방법
- 소요 시간

총 수업 시간은 100분(50분 x 2차시)입니다.
```

✨ **활용 팁:**
- 학생들의 사전 지식 수준을 명시하면 더 적절한 수업이 설계됩니다
- "탐구 활동"이나 "협력 프로젝트"를 강조하면 학생 활동 중심으로 설계됩니다

🔥 **적합한 상황:**
- 새로운 단원 시작 전 체계적인 수업 설계가 필요할 때
- 표준 정렬된 탐구 기반 수업을 계획할 때

### 1.2 단원 설계 (Unit Design)

#### 기본 템플릿

```
당신은 [학교급] [교과] 단원 설계 전문가입니다. [기간] 동안의 [단원명] 단원을 백워드 설계 원리로 개발해주세요.

**대상:** [학년/학급 특성]
**관련 성취기준:** [성취기준 목록]
**핵심 질문:** [단원을 관통하는 본질적 질문]

다음 구조로 설계해주세요:
1. 단원 학습 목표
2. 주차별 학습 내용 및 활동
3. 평가 계획 (형성평가 + 총괄평가)
4. 차시별 핵심 개념 연결 맵
```

#### 예시 프롬프트

📌 **백워드 설계 기반 국어 단원 설계**

```
당신은 고등학교 국어과 단원 설계 전문가입니다. 4주(12차시) 동안의 '비평적 읽기와 쓰기' 단원을 백워드 설계 원리로 개발해주세요.

**대상:** 고등학교 1학년, 기초 학력이 다양한 일반계 고등학교
**관련 성취기준:** 
- [10국02-03] 다양한 관점과 방법으로 글을 읽는다.
- [10국03-04] 글의 내용이나 형식에 대한 비평적 의견을 제시한다.

**핵심 질문:** "비평적 읽기는 어떻게 우리의 사고를 확장하는가?"

다음을 포함해주세요:
1. 단원 학습 목표 (지식, 기능, 태도)
2. 12차시 주차별 학습 내용 및 활동
3. 수행평가 과제 및 루브릭
4. 차시별 형성평가 방법
5. 교차 교과 연계 가능성
```

✨ **활용 팁:**
- 핵심 질문을 먼저 설정하면 단원의 일관성이 높아집니다
- 평가 방법을 먼저 요청하고 역순으로 활동을 설계하도록 하면 백워드 설계의 본질에 충실합니다

### 1.3 교과 연계 프로젝트 (Interdisciplinary Planning)

#### 기본 템플릿

```
당신은 교과 연계 프로젝트 기획 전문가입니다. [학년] 학생들을 위한 [기간] 프로젝트를 개발해주세요.

**연계 교과:** [교과1] + [교과2] (+ ...)
**프로젝트 주제:** [실제 세계 문제/탐구 주제]
**최종 산출물:** [학생들이 만들어낼 결과물]

다음을 포함해주세요:
- 교과별 관련 성취기준
- 단계별 활동 계획
- 협력 활동 설계
- 평가 방법 (과정 + 결과)
```

#### 예시 프롬프트

📌 **환경과학 + 국어 융합 프로젝트**

```
당신은 교과 연계 프로젝트 기획 전문가입니다. 중학교 3학년 학생들을 위한 2주 프로젝트를 개발해주세요.

**연계 교과:** 과학(환경) + 국어(설득적 글쓰기)
**프로젝트 주제:** 우리 지역 환경 문제 해결을 위한 정책 제안서 작성
**최종 산출물:** 지역 환경 문제 분석 보고서 + 정책 제안 프레젠테이션

다음을 포함해주세요:
- 각 교과별 관련 성취기준 명시
- 일자별 활동 계획 (과학 수업 + 국어 수업 연계)
- 모둠 협력 활동 설계
- 과정 평가와 결과 평가 루브릭
- 학생 역할 분담 가이드
```

### 1.4 교사 연수 설계 (Professional Development)

#### 기본 템플릿

```
당신은 교사 전문성 개발 설계 전문가입니다. [대상 교사]를 위한 [시간] 연수를 기획해주세요.

**연수 주제:** [구체적인 주제]
**연수 목표:** [참가자가 얻어갈 역량/지식]
**연수 형태:** [워크숍/강의/실습/혼합]

다음을 포함해주세요:
- 세부 일정 및 시간 배분
- 참여형 활동 설계
- 적용 실습 과제
- 후속 지원 방안
```

#### 예시 프롬프트

📌 **AI 수업 활용 교사 연수**

```
당신은 교사 전문성 개발 설계 전문가입니다. 중학교 교사들을 위한 90분 AI 활용 수업 연수를 기획해주세요.

**연수 주제:** Gemini를 활용한 수업 설계와 학생 학습 지원
**연수 목표:** 
- 효과적인 프롬프트 작성 능력 습득
- 교과별 AI 활용 사례 이해
- 실제 수업 적용 계획 수립

**연수 형태:** 실습 중심 워크숍

다음을 포함해주세요:
- 90분 세부 일정 (15분 단위)
- 참여자 실습 활동 3개 이상
- 제공할 자료 목록
- 연수 후 실천 과제
- 후속 학습 공동체 운영 방안
```

### 1.5 학습 시퀀스 설계

#### 예시 프롬프트

📌 **개념 연결 학습 시퀀스**

```
당신은 과학 교육과정 설계 전문가입니다. 중학교 2학년 물리 영역 '힘과 운동' 단원의 5일 학습 시퀀스를 설계해주세요.

**조건:**
- 학생들의 사전 지식이 다양함 (일부는 기초 개념 부족)
- 매일 핵심 개념 간 연결이 명확해야 함
- 탐구 활동, 협력 학습, 개별 활동이 균형있게 포함

다음 형식으로 제시해주세요:
| 일차 | 핵심 개념 | 주요 활동 | 형성평가 | 다음 차시 연결 |
|------|----------|----------|----------|---------------|
```

## 한국 교육과정 연계 가이드

### 성취기준 표기법

```
[학년군-교과-영역-번호] 성취기준 내용

예시:
- [9국02-03] → 중학교 국어 읽기 영역 3번
- [10영01-04] → 고등학교 영어 듣기 영역 4번
- [6과08-02] → 초등학교 5-6학년 과학 지구와 우주 2번
```

### 2022 개정 교육과정 핵심역량 연계

프롬프트에 다음 역량을 명시하면 역량 중심 수업이 설계됩니다:

| 역량 | 수업 설계 반영 방법 |
|------|-------------------|
| 자기관리 | 학습 목표 설정, 자기 점검 활동 |
| 지식정보처리 | 자료 분석, 정보 평가 활동 |
| 창의적 사고 | 열린 문제, 프로젝트 기반 과제 |
| 심미적 감성 | 예술 작품 감상, 창작 활동 |
| 협력적 소통 | 모둠 활동, 토론, 발표 |
| 공동체 | 지역사회 연계, 시민 참여 프로젝트 |

## 프롬프트 품질 체크리스트

수업 설계 프롬프트 제공 전 확인:

- [ ] 학교급과 학년이 명시되어 있는가?
- [ ] 구체적인 성취기준이 포함되어 있는가?
- [ ] 수업 모형이나 설계 원리가 명시되어 있는가?
- [ ] 수업 시간/기간이 적절하게 설정되어 있는가?
- [ ] 평가 방법이 포함되어 있는가?
- [ ] 한국 교실 맥락(시간, 학급 규모 등)이 반영되어 있는가?

---

## 후속 프롬프트 제안

수업 설계 프롬프트 제공 후 다음을 제안:

1. **자료 제작**: "이 수업에 필요한 학습지를 만들어주세요"
2. **평가 개발**: "이 수업의 형성평가 문항을 개발해주세요"
3. **차별화**: "이 수업을 학습 수준별로 조정해주세요"
4. **학생 활동**: "학생들이 사용할 탐구 가이드를 만들어주세요"
