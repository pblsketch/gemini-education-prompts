# 3. 수업 자료 (Instructional Materials) 프롬프트

학습지, 읽기 자료, 실험 가이드, 토론 자료, 스킬 연습 활동 개발을 위한 프롬프트 가이드입니다.

## 핵심 원칙

### 수업 자료 프롬프트의 PARTS 적용

| 요소 | 수업 자료 맥락에서의 적용 |
|------|--------------------------|
| **P: Persona** | 학습 자료 개발자, 실험 설계 전문가, 콘텐츠 편집자 |
| **A: Act** | 생성하다, 수정하다, 수준 조정하다, 요약하다 |
| **R: Recipient** | 특정 학년/읽기 수준/학습 특성의 학생 |
| **T: Theme** | 학습 주제, 교과 내용, 활동 목적 |
| **S: Structure** | 학습지 형식, 실험 절차, 토론 가이드 형식 |

## 프롬프트 유형별 가이드

### 3.1 읽기 자료 수준 조정 (Re-level Reading)

#### 기본 템플릿

```
당신은 교과 읽기 자료 전문가입니다. 다음 텍스트를 [대상 학생]에게 적합한 수준으로 재구성해주세요.

**원본 텍스트:** [첨부 또는 입력]
**대상 학년:** [학년]
**수준 조정 유형:** [쉽게/어렵게/다중 수준]
**유지 사항:** [핵심 개념, 내용 정확성]

다음을 포함해주세요:
- 조정된 텍스트
- 어휘 설명 (필요시)
- 이해 확인 질문 2-3개
```

#### 예시 프롬프트

📌 **다중 수준 읽기 자료 개발**

```
당신은 교과 읽기 자료 개발 전문가입니다. 첨부된 '광합성' 관련 과학 지문을 중학교 1학년 학생들을 위해 3가지 읽기 수준으로 재구성해주세요.

**원본 텍스트:** [첨부]
**대상 학년:** 중학교 1학년 (다양한 읽기 수준)

**수준 조정:**
- 기초 수준: 초등 5-6학년 수준, 짧은 문장, 그림 설명 포함 제안
- 표준 수준: 중학교 1학년 수준, 교과서와 유사한 난이도
- 심화 수준: 중학교 2-3학년 수준, 추가 개념과 연결

**모든 버전에 포함:**
- 핵심 개념은 동일하게 유지
- 핵심 어휘 목록 (5-7개)
- 이해 확인 질문 2개
- 어휘 난이도 표시
```

✨ **활용 팁:**
- "학생 친화적 언어"를 강조하면 교과서 문체보다 친근한 톤이 됩니다
- "시각적 단서 제안"을 요청하면 그림, 도표 삽입 위치가 표시됩니다

🔥 **적합한 상황:**
- 학급 내 읽기 수준 격차가 클 때
- 동일 개념을 다양한 학생에게 전달해야 할 때

### 3.2 학생용 학습지 (Student Handout)

#### 기본 템플릿

```
당신은 [교과] 학습지 개발 전문가입니다. [학년] 학생들을 위한 [주제/기능] 학습지를 만들어주세요.

**학습 목표:** [구체적인 학습 목표]
**학습지 유형:** [개념 정리/연습 문제/탐구 활동/자기 점검]
**분량:** [A4 몇 장]
**소요 시간:** [예상 소요 시간]

다음을 포함해주세요:
- 학생용 안내문
- 활동 내용
- 자기 점검 요소
```

#### 예시 프롬프트

📌 **개념 정리 학습지**

```
당신은 중학교 국어 학습지 개발 전문가입니다. 중학교 2학년 학생들을 위한 '문학 작품의 관점 분석' 학습지를 만들어주세요.

**학습 목표:** 
- 문학 작품에 나타난 서술자의 관점을 파악할 수 있다
- 관점에 따른 효과를 설명할 수 있다

**학습지 유형:** 개념 정리 + 적용 연습
**분량:** A4 2장 (양면 1장)
**소요 시간:** 30분

다음을 포함해주세요:
1. 핵심 개념 정리 (서술 관점의 유형과 특징)
2. 짧은 예문과 관점 분석 연습 (3개)
3. 스스로 분석하는 적용 문제 (1개)
4. 자기 점검 체크리스트
5. 선생님 피드백 공간

※ 한글 파일에 바로 붙여넣을 수 있는 형식으로 제공해주세요.
```

### 3.3 실험/실습 가이드 (Lab Guide)

#### 기본 템플릿

```
당신은 [교과] 실험 설계 전문가입니다. [학년] 학생들을 위한 [실험/실습명] 가이드를 만들어주세요.

**실험 목적:** [학습 목표와 연결된 실험 목적]
**소요 시간:** [예상 시간]
**준비물:** [필요한 재료/도구]

다음을 포함해주세요:
- 학생용 실험 안내서
- 안전 수칙
- 단계별 절차
- 관찰/기록 양식
- 분석 질문
```

#### 예시 프롬프트

📌 **탐구 실험 가이드**

```
당신은 중학교 과학 실험 설계 전문가입니다. 중학교 2학년 학생들을 위한 'DNA 추출 실험' 가이드를 만들어주세요.

**성취기준:** [9과08-01] 유전 물질이 DNA임을 알고, DNA의 구조적 특성을 이해한다.
**실험 목적:** 실제 생물 시료에서 DNA를 추출하여 유전 물질의 존재를 확인한다
**소요 시간:** 50분 (1차시)
**준비물:** 바나나, 식염수, 주방세제, 에탄올, 비커, 막자사발 등

다음을 포함해주세요:
1. 실험 개요 및 배경 지식 (200자 이내)
2. ⚠️ 안전 수칙 (에탄올 취급 주의 등)
3. 단계별 절차 (10단계 이내, 예상 시간 포함)
4. 예상 결과 및 관찰 포인트
5. 관찰 기록표 (양식)
6. 탐구 질문 3개 (결과 해석 + 심화)
7. 가정에서 할 수 있는 추가 탐구 제안
```

✨ **활용 팁:**
- "탐구 중심"을 강조하면 결과 예측, 가설 설정 단계가 포함됩니다
- "오류 가능성"을 요청하면 흔한 실수와 해결책이 포함됩니다

### 3.4 토론 가이드 (Discussion Guide)

#### 기본 템플릿

```
당신은 [교과] 토론 수업 설계 전문가입니다. [학년] 학생들을 위한 [주제] 토론 가이드를 만들어주세요.

**토론 유형:** [찬반 토론/세미나/하브루타/원탁 토론]
**주제/질문:** [토론 주제 또는 핵심 질문]
**소요 시간:** [예상 시간]
**참여 인원:** [모둠 구성]

다음을 포함해주세요:
- 배경 자료 요약
- 토론 질문 세트
- 역할 배분 가이드
- 토론 진행 절차
- 성찰 질문
```

#### 예시 프롬프트

📌 **하브루타 토론 가이드**

```
당신은 고등학교 사회 토론 수업 설계 전문가입니다. 고등학교 2학년 학생들을 위한 '기본소득제 도입' 하브루타 토론 가이드를 만들어주세요.

**토론 유형:** 짝 하브루타 → 모둠 공유
**핵심 질문:** "기본소득제는 사회적 불평등 해소에 효과적인가?"
**소요 시간:** 50분 (1차시)
**참여 구조:** 2인 짝 → 4인 모둠 → 전체 공유

다음을 포함해주세요:
1. 배경 자료 요약 (찬성/반대 입장 각 3가지 근거)
2. 짝 토론 질문 3개 (5분씩)
3. 모둠 토론 질문 2개 (10분)
4. 전체 공유 발표 가이드
5. 개인 성찰 질문 2개
6. 토론 예절 안내
7. 교사용 진행 시나리오
```

### 3.5 기능 연습 활동 (Skill Practice)

#### 예시 프롬프트

📌 **글쓰기 기능 연습**

```
당신은 중학교 국어 학습 활동 개발 전문가입니다. 중학교 2학년 학생들의 '근거를 들어 주장하는 글쓰기' 기능 연습 활동을 만들어주세요.

**성취기준:** [9국03-03] 관찰, 조사, 실험의 절차와 결과가 드러나게 글을 쓴다.
**연습 목표:** 근거의 적절성과 논리적 연결 연습
**소요 시간:** 30분

다음을 포함해주세요:
1. 기능 설명 (근거 제시의 원리)
2. 모범 예시 (좋은 근거 vs 약한 근거 비교)
3. 단계별 연습
   - 1단계: 주장에 맞는 근거 선택하기
   - 2단계: 근거를 주장과 연결하는 문장 쓰기
   - 3단계: 짧은 주장 문단 완성하기
4. 자기 점검 체크리스트
5. 동료 피드백 양식
```

### 3.6 콘텐츠 개인화 (Content Personalization)

#### 예시 프롬프트

📌 **학생 관심사 반영 자료**

```
당신은 학습 콘텐츠 개인화 전문가입니다. 다음 수학 개념 설명을 학생들의 다양한 관심사에 맞게 재구성해주세요.

**원본 개념:** 일차함수의 기울기와 y절편
**대상:** 중학교 2학년

다음 관심 분야별로 같은 개념을 설명하는 자료를 만들어주세요:
1. **스포츠 관심 학생:** 농구 자유투 성공률, 축구 골 득점 등
2. **게임 관심 학생:** 게임 레벨업, 점수 획득 등
3. **음악 관심 학생:** 음악 스트리밍 순위, 앨범 판매량 등
4. **환경 관심 학생:** 탄소 배출량, 재활용률 등

각 버전에 포함:
- 맥락화된 개념 설명 (200자)
- 관련 예제 문제 2개
- 실생활 연결 질문 1개
```

## 프롬프트 품질 체크리스트

수업 자료 프롬프트 제공 전 확인:

- [ ] 대상 학년과 수준이 명시되어 있는가?
- [ ] 학습 목표가 명확한가?
- [ ] 자료 유형과 형식이 구체적인가?
- [ ] 분량과 소요 시간이 적절한가?
- [ ] 학생용 자기 점검 요소가 포함되어 있는가?
- [ ] 한글 파일 호환 형식이 요청되어 있는가?

## 후속 프롬프트 제안

수업 자료 프롬프트 제공 후 다음을 제안:

1. **수준 조정**: "이 자료를 기초 수준 학생용으로 조정해주세요"
2. **평가 연계**: "이 활동의 평가 기준표를 만들어주세요"
3. **교사 가이드**: "이 자료 활용을 위한 교사용 안내서를 만들어주세요"
4. **디지털 변환**: "이 활동을 온라인 협업 도구용으로 변환해주세요"
